
# GitHub Pages Ready Portfolio

This portfolio has been configured for direct deployment to GitHub Pages.

## Features
- Next.js static export
- GitHub Actions auto deployment
- No Vercel required
- Ready for GitHub repository upload

## Deploy Instructions

1. Create a new GitHub repository
2. Upload all project files
3. Push to the `main` branch
4. Open:
   Settings → Pages
5. Under Source:
   - Select **GitHub Actions**

Your portfolio will automatically build and deploy.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
